A simple multi-threaded map reduce library in python.

To use:
a) have a function by the name of mapper in mapper.py
b) a function by the name of reducer in reducer.py
c)have all the data files in data_folder
d) start as many workers as you want from different terminals by executing python worker.py
e)start the master by python master.py
f)the results are obtained in results.txt file.
